# Healthcare Cybersecurity ML Project

## Overview
This project demonstrates how machine learning can detect suspicious login activity in healthcare systems.

## Features
- Login behavior analysis
- Suspicious activity detection
- Risk scoring system
- Visualization of login patterns

## Machine Learning Model
Decision Tree Classifier used for classification of suspicious behavior.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Google Colab

## Example Output
Risk Score: 80%
Reasons:
- Unusual login time
- High data download
- Unknown device

## Future Improvements
- Real-time monitoring system
- Advanced anomaly detection
